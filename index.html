About VRChat

VRChat is a dynamic social VR experience where creativity meets community. Dive into immersive worlds, customize your avatar, and interact with friends from around the globe – all within a creative, ever-evolving virtual space.

One of VRChat’s standout features is its versatility in loading content. Media such as images, videos, and interactive web pages are loaded directly into the world via stringloading, and advanced scripting allows for data to be parsed from CSV, JSON, and more.

About VRCHost

VRCHost is a private CDN (Content Delivery Network) designed to host files for custom VRChat worlds and experiences. VRChat can load images, videos, audio, and even complex data files from external sources like this one to bring your virtual environments to life.

If you're seeing this page outside of VRChat, something isn’t quite right – either the content didn’t load as expected or the URL was mistyped. This page is intended solely for VRChat use.

This CDN is hosted on GitHub Pages, leveraging a trusted platform to serve static content reliably. The URLs remain permanent and stable.

How Content Works in VRChat

In VRChat, media is loaded directly into the world without the need for an embedded browser framework. This means that all supported media – including images, videos, and data files – can be seamlessly integrated via URLs.

Additionally, VRChat supports stringloading and has powerful scripting capabilities that allow you to load and parse CSV, JSON, and other data formats to enhance your world.
